# Cyberbullying Detection on Social Media

## Overview

This project aims to detect cyberbullying and offensive content on social media platforms using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes user comments and classifies them as bullying or non-bullying based on the trained model.

## Objectives

* Detect harmful and offensive comments.
* Apply NLP techniques for text preprocessing.
* Train a machine learning model for text classification.
* Provide a simple interface for users to test comments.

## Dataset

* Social media comments dataset
* Approximately 27,000 records
* Contains comment text and corresponding class labels
